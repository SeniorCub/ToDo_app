API Documentation

Base URL: http://localhost:9000/api/user

Register User

Endpoint: POST /create

Request Body:

{
    "fullname": "John Doe",
    "email": "john@example.com",
    "photoUrl": "https://example.com/photo.jpg"
}

Response:

{
    "message": "User created successfully.",
    "token": "JWT_TOKEN",
    "data": { "id": 1, "fullname": "John Doe", "email": "john@example.com" }
}
Login User

Endpoint: POST /login

Request Body:

{
    "email": "john@example.com"
}

Response:

{
    "message": "User logged in successfully.",
    "token": "JWT_TOKEN",
    "data": { "id": 1, "fullname": "John Doe", "email": "john@example.com" }
}
Check if User Exists

Endpoint: GET /check?email=john@example.com

Response:

{
    "exists": true
}
Get User Details

Endpoint: GET /:id

Headers: Authorization: Bearer JWT_TOKEN

Response:

{
    "message": "User found successfully.",
    "data": { "id": 1, "fullname": "John Doe", "email": "john@example.com" }
}
Delete User

Endpoint: DELETE /:id

Headers: Authorization: Bearer JWT_TOKEN

Response:

{
    "message": "User deleted successfully."
}
